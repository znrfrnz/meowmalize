  <name>Task 3: StudySession component + /flashcards route</name>
  <files>src/components/StudySession.tsx, src/app/flashcards/page.tsx</files>
  <action>
    **src/components/StudySession.tsx** — Client component managing full study session state:

    ```typescript
    'use client'
    ```

    Internal state (useState):
    - `currentIndex: number` — 0-based index into filtered/shuffled deck
    - `isFlipped: boolean` — whether current card is flipped
    - `filter: 'all' | 'definition' | 'enumeration'`
    - `deck: Flashcard[]` — current working deck (filtered + possibly shuffled)
    - `sessionComplete: boolean` — true when all cards in deck viewed

    On mount: initialize `deck` from `cards` prop filtered by `filter`.

    When filter changes: reset `deck` to filtered subset, reset `currentIndex` to 0, reset `isFlipped` to false.

    **Navigation logic:**
    - `goNext()`: if `currentIndex < deck.length - 1` → increment; else if all cards viewed → `setSessionComplete(true)`
    - `goPrev()`: if `currentIndex > 0` → decrement; reset `isFlipped` to false on navigation
    - `flip()`: toggle `isFlipped`

    **Keyboard handler** (useEffect, cleanup on unmount):
    - ArrowRight → `goNext()`
    - ArrowLeft → `goPrev()`
    - Space → `flip()` (preventDefault)
    - K → `markKnown(currentCard.id)` then `goNext()`
    - L → `markLearning(currentCard.id)` then `goNext()`

    **Shuffle**: Fisher-Yates on deck copy, reset index to 0, reset flipped.

    **Reset progress**: call `resetProgress()` from store — show confirmation first (window.confirm is acceptable for v1).

    **Store integration**: `const { progress, markKnown, markLearning, resetProgress } = useFlashcardStore()`

    **Completion screen** (when `sessionComplete`):
    - Trophy icon (lucide, 48px, accent color)
    - "Deck Complete!" heading
    - Stats: "{knownCount} Known · {learningCount} Still Learning"
    - "Review Still Learning" button: filter to 'still learning' cards, reset session (only if learningCount > 0)
    - "Reshuffle All" button: reset filter to 'all', shuffle, restart

    **Layout structure:**
    ```
    <div className="max-w-2xl mx-auto px-4 py-8 flex flex-col gap-6">
      {/* Filter + Control Bar */}
      {/* Progress indicator */}
      {/* FlashCard or Completion Screen */}
      {/* Action buttons (Know it / Still learning) */}
      {/* Keyboard shortcut legend */}
    </div>
    ```

    **Filter bar** — pill toggle buttons: "All" | "Definitions" | "Enumerations"
    Active pill: `bg-[#6366f1] text-white`; inactive: `bg-[#27272a] text-[#71717a]`

    **Progress indicator**: `"Card {currentIndex + 1} of {deck.length}"` in `font-mono text-sm text-[#71717a]`
    Also show: `"{knownCount} Known"` (green) · `"{learningCount} Learning"` (amber) — small pill counts

    **Action buttons:**
    - "Know it": `bg-[#22c55e] text-white h-11 rounded-lg flex items-center gap-2 px-4`
      - CheckCircle icon (lucide, 18px) + "Know it"
      - onClick: `markKnown(currentCard.id); goNext()`
    - "Still learning": `bg-[#f59e0b] text-white h-11 rounded-lg flex items-center gap-2 px-4`
      - RotateCcw icon (lucide, 18px) + "Still learning"
      - onClick: `markLearning(currentCard.id); goNext()`

    **Keyboard legend** (below action buttons, small and muted):
    ```
    ← → Navigate  |  Space Flip  |  K Know it  |  L Still learning
    ```
