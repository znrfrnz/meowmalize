Create src/stores/deckStore.ts. Pattern: exactly like flashcardStore — 'use client', create + persist + skipHydration: true.

Export constant `BUILTIN_DECK_ID = 'builtin-flashcards'`.

State interface:
```typescript
interface DeckState {
  folders: Folder[]
  decks: Deck[]
  // Folder CRUD
  addFolder: (name: string) => Folder
  renameFolder: (id: string, name: string) => void
  deleteFolder: (id: string) => void  // moves its decks to ungrouped (folderId: null), does NOT delete them
  // Deck CRUD
  addDeck: (name: string, folderId: string | null, cards?: Flashcard[]) => Deck
  renameDeck: (id: string, name: string) => void
  deleteDeck: (id: string) => void  // noop if deck.locked === true
  addCardToDeck: (deckId: string, card: Flashcard) => void
  removeCardFromDeck: (deckId: string, cardId: string) => void
  updateCardInDeck: (deckId: string, card: Flashcard) => void
  setDeckCards: (deckId: string, cards: Flashcard[]) => void
}
```

Initial state:
- `folders: []`
- `decks: [BUILTIN_DECK]` where BUILTIN_DECK is:
  ```typescript
  const BUILTIN_DECK: Deck = {
    id: BUILTIN_DECK_ID,
    name: 'Database Concepts',
    folderId: null,
    cards: FLASHCARDS,
    createdAt: '2026-01-01T00:00:00.000Z',
    locked: true,
  }
  ```

Key behaviors:
- `addFolder`: generates id with `crypto.randomUUID()`, returns the new Folder
- `addDeck`: generates id with `crypto.randomUUID()`, returns the new Deck
- `deleteFolder(id)`: set all decks with folderId===id to folderId=null, then remove folder
- `deleteDeck(id)`: if decks.find(d=>d.id===id)?.locked is true, return without doing anything

Persist config:
```typescript
persist(..., {
  name: 'infoman-decks',
  skipHydration: true,
})
```
