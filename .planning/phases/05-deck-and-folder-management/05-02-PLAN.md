  <action>
Create src/app/decks/page.tsx as a 'use client' component.

**Layout:**
- Page header: "My Decks" h1 + "+ New Folder" button (top right)
- Below: folders listed as accordion sections (expanded by default), then an "Ungrouped" section for decks with folderId=null (excluding the built-in deck which shows separately as "Built-in")

**Folder section:**
- Section header: folder name + deck count + "+" (add deck) + trash icon (delete folder)
- On "+ New Deck" click: show inline input for deck name, press Enter or click ✓ to create via `addDeck(name, folderId)`
- On folder delete: call `deleteFolder(id)` — moves its decks to ungrouped, confirm with a simple `window.confirm`

**Deck card (inside a folder or in ungrouped):**
- Shows deck name + card count
- Lock icon (🔒 or Lucide `Lock`) if deck.locked, no delete button for locked decks
- Trash icon for non-locked decks — calls `deleteDeck(id)` with `window.confirm`
- Clicking the deck name or card navigates to `/decks/${deck.id}` via `<Link>`

**"+ New Folder" flow:**
- Toggles an inline input at top of folder list
- Press Enter or click ✓ → calls `addFolder(name)`, clears input

**"Built-in" section:**
- Always rendered last, non-collapsible, contains only the built-in deck
- Visual treatment: slightly different border color (e.g. `border-[#6366f1]/20`) to distinguish

Use Lucide icons: `Lock`, `Trash2`, `Plus`, `FolderOpen`, `ChevronDown`.

Keep consistent with existing design: bg-[#0f0f0f], surface bg-[#1a1a1a], borders border-[#27272a], accent #6366f1, muted #71717a.
